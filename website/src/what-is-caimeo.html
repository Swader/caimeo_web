What is Caimeo?

An Open Source Framework for Sovereign AI Agents

Caimeo is an open-source framework designed for creating and managing self-sufficient, sovereign AI agents that operate without human intervention. Whether running on-chain as autonomous entities or off-chain in specialized simulation environments, Caimeo provides the foundation for next-generation AI systems.

Core Features

  • Fully autonomous agent operations
  • Sovereign identity management
  • Cross-chain communication
  • Simulation environment support

Key Benefits

  • Zero human intervention required
  • Flexible deployment options
  • Modular architecture
  • Enhanced privacy and security

Technical Foundation

Built on established standards for maximum compatibility and security

ERC-6220: Composable NFTs

The backbone of Caimeo's modular agent system. This standard enables:

  • Nested NFT structures for complex agent compositions
  • Dynamic equipment slots for skills and traits
  • Hierarchical relationships between agent components

ERC-6454: Soulbound 2.0

Powers non-transferable core agent attributes:

  • Permanent personality traits
  • Learned behaviors and experiences
  • Core intelligence modules

ERC-5773: Multi-Asset

Enables agent evolution and state changes:

  • Seamless evolution from Soul Shard to full agent (learn more)
  • Multiple visual and functional states
  • Progressive unlocking of capabilities

Core Framework Integration

Caimeo combines the best of two leading frameworks:

ElizaOS - Agent Operations

  • Multi-model AI support (Llama, OpenAI, Anthropic)
  • Built-in memory and document store
  • Proven autonomous agent operations

Freysa (FAI) - Sovereignty Layer

  • Secure TEE-based key management
  • Advanced state persistence
  • Committee-based governance

By combining these frameworks, Caimeo achieves both operational excellence and true agent sovereignty, allowing developers to choose the components that best suit their needs.
